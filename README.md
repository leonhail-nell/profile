# Personal Profile Website

A Next.js + MUI portfolio starter with a modern profile layout.

## What’s included

- `package.json` — Next.js and MUI dependencies
- `app/layout.tsx` — app router root layout with theme provider
- `app/page.tsx` — profile landing page with hero, about, work, skills, and contact
- `theme.ts` — custom MUI theme configuration
- `app/globals.css` — global styles for the app

## Setup

1. Open a terminal in this folder.
2. Run `npm install`.
3. Run `npm run dev`.
4. Open `http://localhost:3000`.

## Customize

- Replace placeholder text with your name, story, and project details.
- Update `theme.ts` to change colors and typography.
- Add more sections or personal content in `app/page.tsx`.

## Deployment

Deploy on Vercel, Netlify, or any platform that supports Next.js.
